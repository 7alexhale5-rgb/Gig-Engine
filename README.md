# PrettyFly Acquisition Engine

Centralized operations platform for managing freelance project acquisition across Upwork, Fiverr, Toptal, Arc.dev, and other gig economy platforms.

Built with Next.js 14 (App Router), TypeScript, Tailwind CSS, shadcn/ui, and Supabase.

## Quick Start

```bash
npm install
cp .env.local.example .env.local  # Fill in Supabase credentials
npm run dev
```

## Stack

- **Frontend:** Next.js 14, TypeScript, Tailwind CSS, shadcn/ui
- **Backend:** Supabase (PostgreSQL, Auth, Edge Functions)
- **Charts:** Recharts
- **Drag-and-Drop:** @dnd-kit
- **Forms:** React Hook Form + Zod
