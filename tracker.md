## Morning Workflow Protocol

**Daily Schedule: 7:00 - 7:45 AM**

This is the highest-ROI 45 minutes of the day. Consistency compounds -- 5-10 proposals per day = 35-70 per week = 150-300 per month. At even a 5% win rate, that is 7-15 new contracts monthly.

### Step 1: Upwork Saved Searches (7:00 - 7:10)

1. Open Upwork and check all 5 pillar saved searches (see table above)
2. Star any jobs posted in the last 24 hours that match your services
3. Prioritize: jobs posted <6 hours ago, <10 proposals already, budget >$500
4. Flag 5-10 best-fit jobs for proposals

### Step 2: Generate Proposals (7:10 - 7:30)

1. For each flagged job, run the `proposal-generator` agent:
   - Input: job posting URL or description
   - Select matching template from listing's sample proposal
   - Agent generates personalized proposal draft
2. Review and customize each draft (30-60 seconds per proposal)
3. Replace all [BRACKETED] placeholders with job-specific details
4. Ensure each proposal includes:
   - Specific reference to the client's problem
   - Relevant portfolio piece or case study
   - Clear deliverables and timeline
   - Competitive price (reference Upwork listing pricing)
5. Submit 5-10 proposals

### Step 3: Track Proposals (7:30 - 7:35)

1. Update the Upwork Listings table above:
   - Increment `Proposals Sent` count for each relevant service
   - Log any interview invites or responses from yesterday
   - Update `Contracts` and `Revenue` for any new wins
2. Note any patterns: which service pillars are getting traction?

### Step 4: Fiverr Check (7:35 - 7:40)

1. Open Fiverr seller dashboard
2. Check for new messages -- respond within 1 hour (Fiverr response time metric)
3. Check for new orders -- acknowledge and set expectations immediately
4. Review gig analytics: impressions, clicks, conversion rates
5. Update Fiverr Listings table above with fresh metrics

### Step 5: Update Tracker & Log (7:40 - 7:45)

1. Update the Deployment Status header counts if any listings changed status
2. Add any notable events to the Notes column of relevant tables
3. Log total time spent on morning workflow
4. If it is Monday, fill in the Weekly Performance Summary row for the prior week

### Morning Workflow Checklist (Daily)

- [ ] Checked all 5 Upwork saved searches
- [ ] Flagged 5-10 matching jobs
- [ ] Generated and submitted 5-10 proposals
- [ ] Updated Upwork proposal counts in tracker
