# Proposal Template Library

## Template Structure

Every proposal follows this framework (under 200 words):

```
HOOK — Restate their problem proving you read the posting
CREDIBILITY — One sentence, concrete result
APPROACH — 3-4 bullet deliverables + timeline
DIFFERENTIATOR — Why you over the other 30 proposals
CTA — Direct, with specific start timeline
```

## Templates

### 1. GHL Automation (Upwork)
**Use for:** GoHighLevel setup, CRM automation, workflow builds

> "You need your GoHighLevel instance actually working for you — not just sitting there with half-built workflows. I specialize in exactly this."
>
> "I've built and optimized GHL systems for 15+ service businesses, including full pipeline automation, lead routing, and review generation sequences."
>
> - Audit your current GHL setup and identify automation gaps
> - Build/rebuild workflows for [their specific needs]
> - Configure lead scoring, pipeline triggers, and notification systems
> - Full documentation + 30-min walkthrough call
>
> "I also serve as CTO for a construction enterprise, so I bring enterprise-level systems thinking to every GHL build."
>
> "Happy to jump on a 15-min call to scope this. I can start this week."

### 2. n8n/Automation Workflow (Fiverr + Upwork)
**Use for:** Any workflow automation, system integration, API connections

> "You need [System A] talking to [System B] without manual data entry. I build these integrations daily."
>
> - Map complete data flow between systems
> - Build production-ready workflow with error handling
> - Test with real data + edge cases
> - Documentation and monitoring setup
>
> "I build with n8n (self-hosted) so you own the infrastructure — no per-task fees like Zapier."

### 3. AI Chatbot/Assistant (Fiverr + Upwork)
**Use for:** AI chatbot deployment, custom assistants, lead qualification bots

> "You want an AI assistant that actually understands your business — not a generic chatbot."
>
> - Define conversation flows and knowledge base
> - Build and train on your business context
> - Integrate with CRM/website/channels
> - Test, refine, deploy with monitoring
>
> "I'm not just plugging in ChatGPT — I architect AI systems that integrate with your existing tools."

### 4. CRM Migration (Upwork)
**Use for:** CRM platform migrations, data migration, system transitions

> "Migrating CRMs is painful when done wrong and invisible when done right."
>
> - Audit source system data structure and quality
> - Map fields, relationships, and automation logic
> - Execute with validation at every step
> - Rebuild automations in new system
> - Parallel run period to confirm accuracy

### 5. Custom Dashboard Build (Upwork + Fiverr)
**Use for:** Executive dashboards, operations panels, reporting tools

> "You need to see your business metrics in one place instead of logging into 5 different tools."
>
> - Define KPIs and data sources
> - Design with your priorities
> - Build real-time data connections
> - Deploy to your domain with auth
>
> "This isn't a spreadsheet with charts. It's a custom web app, real-time, any device."

### 6. Landing Page (Fiverr)
**Use for:** High-conversion landing pages, lead gen pages

> "Your landing page has 3 seconds to prove visitors should stay."
>
> - Conversion-focused copy and layout
> - Mobile-first responsive
> - Fast loading (<2 sec)
> - Form integration with CRM/email
>
> "Production-grade code, not drag-and-drop. Fast, SEO-ready, fully yours."

### 7. Tech Stack Audit (Upwork)
**Use for:** System audits, tool consolidation, tech strategy

> "You're paying for 8-12 tools that don't talk to each other."
>
> - Inventory all tools, costs, usage
> - Map data flows, identify gaps/redundancies
> - Consolidation roadmap with priorities
> - Specific replacements with migration plan
> - ROI projection
