# Platform-Specific Optimization Guides

## Upwork

### Profile Optimization
- Headline: keyword-dense, 70 chars max
- First 250 chars of overview are critical (shown in search)
- 5+ portfolio case studies with screenshots and metrics
- 2-3 Specialized Profiles (Automation, AI, CRM)
- Hourly rate: $95-125 (premium but accessible)
- Respond within 2 hours (algorithm tracks this)

### Proposal Strategy
- Apply within first 5-15 proposals to a posting
- Daily cadence: 7:00-7:45 AM, 5-10 proposals
- Keep under 200 words: Hook, Credibility, Approach, Differentiator, CTA
- Use boosted proposals on high-value jobs only

### Top Rated Path
- Week 1-4: Land first 2-3 contracts
- Week 4-8: Rising Talent badge
- Month 3-4: Top Rated eligibility ($1K+ earned, 90%+ JSS)

## Fiverr

### Algorithm Factors (ranked)
1. Relevance (title, tags, description keywords)
2. Performance history (completion rate, on-time, response time)
3. Seller level progression
4. Recency (active sellers boosted)
5. Conversion rate (impressions → orders)
6. Review volume and recency
7. Buyer satisfaction metrics

### Optimization Tactics
- Update gig descriptions weekly (signals "active")
- 100% response rate within first hour
- Deliver early when possible
- Start with lower pricing for review velocity, then raise
- Promoted Gigs on top performers only ($10-15/day)
- Never cancel orders — negotiate scope adjustments
